Mython

Простой язык программирования с GUI
Пишешь — запускаешь. Никаких лишних настроек.

GUI окна Картинки Физика Клавиши VS Code
📝

Простой синтаксис

Переменные, условия, циклы, функции — всё как в Python, но проще.

🖥️

GUI

Одна строка window() — и окно готово. Кнопки, фон, текст.

🎮

Игровая физика

Гравитация, прыжки, блоки, коллизии — пиши игры без движка.

⌨️

Управление

Любая клавиша — обработчик. Держишь — плавно двигается.

🔌

VS Code плагин

Подсветка, кнопка ▶, F5 для запуска. Всё уже внутри.

🐍

На Python

Нужен только Python 3. Опционально — Pillow для PNG.

Быстрый старт

# hello.myt
var name = "Mython"

function greet:
    print("Привет из " + name + "!")

window(400, 300, title: "Мой app")
background(color: "#f0f4ff")
button("Поздороваться", onclick: greet, color: "#4a90d9", textcolor: "white")

Прочие

Добавить картинку

# Обычная картинка
image(name: "coin", size: 32, src: "coin.png", x: 200, y: 100)

# С физикой — падает, прыгает
image(name: "player", size: 48, src: "player.png", x: 100, y: 50, physics: true)

# Блок — твёрдый, останавливает физику
image(name: "ground", size: 32, src: "ground.png", x: 0, y: 368, block: true)

Движение и прыжок

move_x("player", 5)   # вправо
move_x("player", -5)  # влево
move_y("player", -5)  # вверх
jump("player")        # прыжок (только на земле)

Коллизии

on_collide("player", "coin"):
    print("Монета!")

press_key("left"):   move_x("player", -5)
press_key("right"):  move_x("player", 5)

    print("Монета собрана!")

Установка

1
Скачай mython-1.0.0.vsix
2
В VS Code: Extensions → ··· → Install from VSIX
3
Открой .myt файл и нажми F5 или нажав кнопку Run Mython File
